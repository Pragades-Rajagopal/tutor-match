# Tutoree Backend Service

Developed using Platformatic DB

## Requirements

- Platformatic 
- Node

## Setup

1. Install dependencies:

```bash
npm install
```

2. Apply migrations:

```bash
npm run migrate
```


## Usage

Run the API with:

```bash
npm start
```
